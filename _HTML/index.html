  • Primeira Aula - Introdução à Programação
    • Introdução

      • Introdução à Programação Programar é a capacidade de dar ordens a um computador, dizendo exatamente o que ele deve fazer. Quando você escreve um programa, está criando uma série de instruções (ou código) que o computador segue para executar tarefas. Esses comandos funcionam como uma receita/manual, onde você explica passo a passo como o computador deve resolver um problema ou completar uma atividade.

        O que é Programação? Imagine que você quer que o computador faça algo, como calcular um número por exemplo. Para isso, você precisa "falar" com ele em uma linguagem que ele entenda.

        A programação é o processo de escrever essas instruções para que o computador as siga. Existem muitas linguagens de programação diferentes, como Python, Java, JavaScript, C++, entre outras. Cada uma tem uma maneira diferente de escrever o código, mas todas têm um objetivo comum: dar instruções ao computador.

        Algoritmos. Quando você escreve um programa, está criando uma sequência de passos para o computador executar. Cada passo é um comando ou uma instrução, o conjunto desses passos/instruções chama se algoritmo.

        Então... Algoritmo é uma sequencia de passos/instruções finita que você utiliza para atingir um resultado ou resolver algo. Basicamente um manual de instrução. A maioria das coisas que compramos hoje vem com uma manual de instrução, explicando como montar, como utilizar... Exemplo montar uma cadeira gamer.

        O computador segue essa sequência de forma rápida e precisa. Se você quiser que o computador faça uma série de ações, como calcular uma soma, mostrar algo em tela você precisará instruí-lo de maneira clara e exata.

        Mas tudo isso deve segui uma logica.

        Lógica de programação É Basicamente a maneira de como damos instrução para o computador de uma forma que ele entenda. O computador e excelente em seguir ordens, por isso precisamos dar instruções muito claras e detalhadas de como ele deve seguir. Quando seguimos o manual não podemos alterar a ordem dos passos, pois vai dar errado. Precisamos seguir de maneira lógica para alcançar o objetivo.

    • Estruturas Condicionais em C
        Operadores em Condicionais

        Para formar condições nas estruturas condicionais, são utilizados operadores como:
        • Operadores relacionais: >, <, >=, <=, ==, !=.
        • Operadores lógicos: && (e lógico), || (ou lógico), ! (negação).

        Exemplo de uso:

        if:
        É a estrutura mais simples de condição,onde permite executar um bloco de código somente se uma condição for verdadeira.

        Sintaxe: if (condição) {

        // Bloco de código executado se a condição for verdadeira

        }

        Exemplo: 

        
    int main() {
        int numero;
        printf("Digite um numero: ");
        scanf("%d", &numero);
        if (numero > 0) {
            printf("O numero é positivo.\n");
        }
        return 0;
    }

        if-else:
        Permite um caminho alternativo, caso a condição seja falsa. Essa estrutura permite executar um bloco de código se uma condição for verdadeira e agora, como alternativa, traz outro bloco de código se a condição for falsa.

        Sintaxe:

        if (condição) {

        // Bloco de código executado se a condição for verdadeira

        } else {

        // Bloco de código executado se a condição for falsa

        }
